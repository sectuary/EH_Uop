> **PT UOP BScCSFC / DSA / CS 17 EH**

## Assessment Overview

| Component | Weight | Format |
|-----------|--------|--------|
| Individual Exercise Report (Labs Week 1-6) | 30% of 35% | Lab reports with screenshots |
| Web Plugin Research | 50% of 35% | Research report |
| Self Development Reflection | 20% of 35% | Reflective essay |
| Group Presentation | 15% | Research & demo a repeating vulnerability in a popular web plugin |
| Exam | 50% | All topics, remote open book |

## Tools You'll Need

- **Kali Linux** (VM or bare metal)
- **VirtualBox** or **VMware** for running vulnerable VMs
- **TryHackMe** account (see registration guide in LMS)
- **Burp Suite Community Edition**
- **Nmap**, **Metasploit**, **Wireshark**, **Gobuster**, **WPScan**, **Nikto**
